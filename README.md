# Comments Project

A comment project is a simple, single-page facebook comment clone.
This project provide users' writing, updating and deleting comments

## Getting Started

- Install dependencies using the `npm install` command.

## Dependencies

- Express
- Node 5.10.x or above
- MongoDB
- React
- Body-parser
- Foreman
- Marked
- Axios
